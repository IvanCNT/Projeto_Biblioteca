# Projeto de Banco de Dados - Sistema de Biblioteca

Este repositório contém os scripts SQL desenvolvidos para a **Experiência Prática IV**. O projeto simula o banco de dados de uma biblioteca universitária, permitindo o cadastro de alunos, livros, funcionários e o controle de empréstimos.

## Estrutura dos Arquivos

* `0_create_tables.sql`: Estrutura do banco de dados (DDL).
* `1_inserts.sql`: Comandos para povoar o banco com dados iniciais (DML).
* `2_selects.sql`: Consultas exemplificando relatórios e junções (JOINs).
* `3_update_delete.sql`: Scripts de atualização cadastral e exclusão de registros.

## Como Executar

1.  Abra seu gerenciador de banco de dados (MySQL Workbench, PGAdmin, etc.).
2.  Crie um banco de dados: `CREATE DATABASE Biblioteca;`
3.  Execute os scripts na ordem numérica (primeiro a criação das tabelas, depois inserts, etc.).

## Tecnologias Utilizadas

* SQL (Structured Query Language)
